Personal Blog

This is a basic personal blog website that allows users to create, edit and delete posts.

Technologies used:

  • Next.js
  • Strapi CMS
  • Emotion (Styled components)
  • Infinite scroll

How to run in a development environment ?

  • cd backend
  • npm install
  • npm run develop
  • cd frontend
  • npm install
  • npm run dev
  • Open http://localhost:3000 in your browser
